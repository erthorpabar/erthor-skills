WorldQuant 公司在2015年发表的论文《101 Formulaic Alphas》 中公开的101个量化Alpha因子。
101 Formulaic Alphas 概述
这101个因子主要基于价格(OHLC)、成交量(Volume)、市值(Cap)等市场数据构建，可以分为以下几大类：
📊 因子分类
类别	        数量	核心思想
动量因子	    ~20个	基于价格趋势延续
反转因子	    ~15个	基于价格均值回归
成交量因子	    ~15个	成交量异常与价格关系
价量关系因子	~20个	价格与成交量的交叉分析
波动率因子	    ~10个	价格波动率特征
相对强弱因子	~10个	相对行业/市场的表现
技术形态因子	~11个	基于K线形态和技术指标


101因子                -> 短期
情绪面(政策)            -> 突发事件
基本面(上下游 + 财报)   -> 长期
