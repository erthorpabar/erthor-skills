# 招聘岗位：大模型应用开发工程师

## 岗位描述

我们正在寻找一位对大语言模型（LLM）技术有深入理解，擅长构建智能应用的开发工程师。你将参与开发基于 LangChain / LangGraph 的多 Agent 系统，推动 RAG 系统、提示词工程（Prompt Engineering）和智能 Agent 工作流的落地。

你将与产品经理、算法工程师和后端工程师紧密合作，共同打造下一代 AI 应用，如智能问答、自动化流程、AI 助手等系统。

## 岗位职责

- 设计和实现基于 LangChain / LangGraph 的大模型智能工作流；
- 构建 RAG 系统，结合企业知识库完成问答、摘要、推荐等任务；
- 设计多智能体（Agent）协作流程，支持复杂任务拆解与并行执行；
- 优化提示词与上下文设计，提升 LLM 的响应质量与一致性；
- 与团队协作，共同推进大模型能力在产品中的落地。

## 任职要求

- 熟练掌握 LangChain、LangGraph 等主流大模型框架；
- 熟悉 Agent 架构、RAG 工作机制、提示词设计与上下文增强方法；
- 精通 Python 编程，具备良好的模块化开发能力；
- 有使用 OpenAI、Claude、LLaMA 等大模型 API 开发经验者优先；
- 具备良好的问题分析、任务建模与文档能力。

## 加分项

- 有过 LangGraph 多 Agent 系统的设计或实战经验；
- 熟悉知识库构建、文本向量化、Embedding 检索等技术；
- 有开源项目或博客/公众号文章可供参考者优先。

## 工作地点

可远程或在北京、上海办公。
